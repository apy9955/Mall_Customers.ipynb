# Customer Segmentation Project

## Overview
This project uses the K-Means clustering algorithm to segment customers based on their Annual Income and Spending Score.

## Technologies Used
- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Data Cleaning
- Feature Scaling
- Elbow Method
- K-Means Clustering
- Customer Segmentation Visualization
- Cluster Analysis

## Dataset
Mall_Customers.csv

## Results
The model identified five distinct customer segments, helping businesses understand customer behavior and improve marketing strategies.
